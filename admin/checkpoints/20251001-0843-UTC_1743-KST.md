## Summary

- `schema-builder`와 `routeStore`를 Surface → Route → Slot 계층 데이터로 확장하고 기본 선택 로직을 정비했다.
- RouteTree UI가 슬롯·섹션 체크박스를 제공하도록 개편하고 스타일을 Surface/Route/Slot별로 재정비했다.
- Execution 패널과 메인 앱에서 새로운 계층 데이터를 사용하도록 갱신했다.
